# Galo Jiu-Jitsu

Site estático responsivo em HTML, CSS e JavaScript.

## Estrutura

- `index.html`
- `css/styles.css`
- `css/schedule.css`
- `js/main.js`
- `assets/`
- imagens e vídeos na raiz do projeto

## Publicação

Para publicar na Hostinger, envie todo o conteúdo desta pasta para o diretório público do domínio, geralmente `public_html`.

O botão de localização abre o link do Google Maps fornecido.
